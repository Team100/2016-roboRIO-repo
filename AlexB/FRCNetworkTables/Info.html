Getting Started

What you need

  • Python 3.x
    • You NEED to get PIP
  • PyCharm CE
  • PyNetworkTables

Installing PyNetworkTables

Once you have Python and PIP installed, update pip with python -m pip install --upgrade pip and then install with pip install pynetworktables

Configuration

IP Configuration

To configure your motors, edit config.py. Look for the line that says:
ip = 'xxx' Change the value depending on your configuration:

  • 'localhost' - Running NTserver on your PC in OutlineViewer Server Mode
  • '172.22.11.2' - Put local IP for USB connection
  • '10.xx.xx.2' - Network IP for competitions


Running the code

  1. Start Eclipse
  2. Download a program
  3. Open PyCharm
  4. Open the NetworkTable project
  5. Configure your setup in config.py
  6. Open FRC Driver Station
  7. Open Shuffleboard
  8. Enable DriverStation
  9. Run main.py in PyCharm or in Command Prompt
  10. Stop when done

If this does not work, check the PyCharm debugger. If it does not say INFO:nt:CONNECTED xxx.xx.xx.x port xxxx (Robot) or similar, a connection to the robot has failed.

Debugging

Connection Failed

If a connection has failed, check that:
  • The IP in config.py is correct
  • The robot has code downloaded
  • The driver station is enabled
  • The computer and RoboRIO are on the same network
You can use Outlineviewer.jar to ensure that your computer can connect to the RoboRIO.
